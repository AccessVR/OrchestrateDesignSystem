# OrchestrateXR Design System — project rules

Persistent brand rules for this project. Follow them in every deliverable.

- **Buttons are ALWAYS uppercase.** Every button label is UPPERCASE (the `Button` component does this by default for all variants except `link`, which is inline sentence-case text). Never ship a sentence-case button. This is a strict brand rule.
- **Headlines:** Mona Sans at **normal width (100%), weight 800**, line-height ~1.2, slight negative tracking (~−1px). Not expanded-width, not weight-900 (too heavy).
- **Ink is warm near-black `#231B15`** — never pure `#000`.
- **Accents:** orange `#FF6600` is primary; **purple is a valid secondary accent**. Keep orange primary; don't let purple into everyday body UI.
- **No emoji** in OrchestrateXR interfaces.
- **Logo:** use the bundled `Logo` (wordmark) / `LogoMark` (icon), with `onDark` on dark surfaces. Header logo renders ~34px tall.
